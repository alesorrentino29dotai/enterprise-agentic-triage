# GPU-only dependencies — install ONLY on the machine that hosts the local LLM
# (e.g. your RTX 5070 + Ryzen AI 9 box). Do NOT add these to the base
# requirements.txt; vLLM pulls a large CUDA PyTorch stack.
#
# RTX 50-series (Blackwell, sm_120): let uv match the CUDA build to your driver.
# Forcing a CUDA version that differs from the vLLM wheel causes
# `ImportError: libcudart.so.<N>`. Use auto-detection in a clean venv:
#
#   uv venv .venv-vllm --python 3.12
#   uv pip install --python .venv-vllm -r requirements-gpu.txt --torch-backend=auto
#
# `--torch-backend=auto` reads `nvidia-smi` and picks the matching wheel
# (e.g. cu130 for a CUDA 13.x driver, cu128 for CUDA 12.8). Pin explicitly with
# --torch-backend=cu130 / cu128 if you must. Verify the GPU first:  nvidia-smi
